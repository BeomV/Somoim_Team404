# Somoim_Team404
멀티캠퍼스 파이널 프로젝트


1. 초기화
```
git init
```

2. 파일 모두 추가
```
git add.
```

3.상태 확인 (선택사항)
```
git status
```

4.Commit
```
git commit -m "Commit Name"
```

5. GitHub Push
```
git push origin 브랜치명
```

6. 브랜치 스위치
```
git checkout 브랜치명
